# Идея: AI-тренер для тренажёрного зала

## Концепция

Создать Telegram-бота, который будет выступать в роли персонального AI-тренера по фитнесу.

## Основные задачи

1.  **Составление персональных тренировочных программ:** Бот должен генерировать программы тренировок на основе данных, предоставленных пользователем.
2.  **База данных упражнений:** В основе бота будет лежать заранее составленная база упражнений и видеоматериалов, сгруппированных по мышечным группам.
3.  **Учет пользовательской информации:** Для персонализации тренировок бот будет запрашивать у пользователя информацию через детальный поток регистрации:
    - Пол (мужской/женский)
    - Возраст (число)
    - Рост (см)
    - Текущий вес (кг)
    - Уровень подготовки (начинающий, 1-3 года, >3 лет)
    - Цель (набор массы, похудение, поддержание формы)
    - Цель по весу (целевой вес в кг)
    - Частота тренировок (2/3/5 раз в неделю)
    - Тип оборудования (зал/свой вес)

## Пользовательский сценарий

1.  Пользователь запускает бота и получает приветственный кружок (video_note), а потом проходит детальный поток регистрации из 10 шагов с приветствием и кнопкой "Начать".
2.  На основе ответов бот составляет индивидуальный план тренировок по дням с описание урпажнений и видеоматериалом к каждому упражени..
3.  В будующем тренер основанный на LLM
